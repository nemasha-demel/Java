-In both public and default access modifiers, we can access classes within the package.
-If the access modifier is public, the class can be accessed by another classes which are in another packages.
-If the access modifier is defaulft, the classes can't be accessed by another classes which are in anothe packages

access modifiers in methods
    -public
    -private
    -defaulft
    -protected

    *public - can be accessed by anyone
    
    *private - only the declared class

    *defaulft - Access within the package

    *protected - Access within the package and subclasses. If the class is inherited from another package,
     the methods and variables which are protected can be accessed.
